# Ben Eater 8-bit Breadboard Computer 

Building a complete 8-bit breadboard computer from scratch, following [Ben Eater's series](https://eater.net/8bit).

## Lerning objectives
- Understanding 8-Bit CPU-architectur 
- 74LS TTL logic circuts 
- Memory (RAM/ROM) & bus systems 
- Clock generation and timing 
- Programming control logic  

## Hardware status 

### Already on Hand (from Fundoino Mega 2560 starter kit):

- Arduino Mega 2560 R3 
- LEDs, switches, resistors, breadboard power supply 

### On order  (full list: doc/parts-list.md)

- 830-point breadboards 
- 74LS ICs (various)
- 7-segment displays, 7447 decoder, NE555 timer
- 28C16 EEPROM  
- 22awg solid core wire, capacitors, resistors
